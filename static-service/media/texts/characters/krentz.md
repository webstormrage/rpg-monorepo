# Krentz
*Средний гуманоид (человек), хаотично-злой*

## Образ
Агрессивный уличный боец и задиристый лидер мелкой банды. Склонен к насилию и давлению.

## Атрибуты
- **Класс Доспеха**: по `bandit`
- **Хиты**: по `bandit`
- **Скорость**: 30 фт.
- **СИЛ / ЛОВ / ТЕЛ / ИНТ / МДР / ХАР**: по `bandit`
- **Навыки**: уличный бой, устрашение
- **Чувства**: пассивная Внимательность по `bandit`
- **Языки**: Common
- **Опасность**: 1/8 (ориентир `bandit`)
- **Специализация**: ближний бой в составе группы
- **Снаряжение**: вооружение и снаряжение по `bandit`

## Известные факты
Имеет связи с Xanathar Guild и действует как полевой силовик.

## Бэкграунд
Криминальный элемент Докового района, привыкший работать в составе банды.
