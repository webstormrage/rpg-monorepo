# Xoblob
*Маленький гуманоид (глубинный гном), мировоззрение не уточнено*

## Образ
Эксцентричный лавочник с яркой фиолетовой эстетикой и странными манерами. Предпочитает торг и мелкие сделки.

## Атрибуты
- **Класс Доспеха**: по социальному NPC-профилю мастера
- **Хиты**: по профилю мастера
- **Скорость**: 25 фт. (типично для svirfneblin)
- **СИЛ / ЛОВ / ТЕЛ / ИНТ / МДР / ХАР**: по профилю мастера
- **Навыки**: торговля, информирование, наблюдательность
- **Чувства**: темное зрение (типично для глубинных гномов)
- **Языки**: Common, Gnomish (и др. по мастеру)
- **Опасность**: социальный NPC
- **Специализация**: информатор/контакт в Доковом районе
- **Снаряжение**: ассортимент безделушек и магазинные товары

## Известные факты
Владеет Old Xoblob Shop и связан с подпольными каналами информации.

## Бэкграунд
После травматичного опыта в Undermountain стал одержим необычными предметами и фиолетовой символикой.
