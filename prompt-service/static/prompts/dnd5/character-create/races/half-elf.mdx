# Выборы для half-elf

На этапе выбора расы `half-elf` нужно сделать 3 выбора.

## Обязательные выборы

### 1. Две характеристики на +1
Харизма уже получает `+2`.
Нужно выбрать ещё 2 другие характеристики, которые получат `+1`.

Ограничения:
- нельзя выбрать `charisma`
- нужно выбрать 2 разные характеристики

Доступные варианты:
- `strength`
- `dexterity`
- `constitution`
- `intelligence`
- `wisdom`

Формат ответа:
`abilityBonuses: [<ability>, <ability>]`

### 2. Два навыка
Нужно выбрать 2 навыка:
- `acrobatics`
- `investigation`
- `athletics`
- `perception`
- `survival`
- `performance`
- `animal handling`
- `intimidation`
- `history`
- `sleight of hand`
- `arcana`
- `medicine`
- `deception`
- `nature`
- `insight`
- `religion`
- `stealth`
- `persuasion`

Формат ответа:
`skillProficiencies: [<skill>, <skill>]`

### 3. Дополнительный язык
Помимо `Common` и `Elvish`, нужно выбрать ещё 1 язык:
- `common`
- `dwarvish`
- `elvish`
- `giant`
- `gnomish`
- `goblin`
- `halfling`
- `orc`
- `abyssal`
- `celestial`
- `draconic`
- `deep-speech`
- `infernal`
- `primordial`
- `sylvan`
- `undercommon`

Формат ответа:
`extraLanguage: <slug>`
