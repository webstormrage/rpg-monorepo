# Выборы для high-elf

На этапе выбора расы `high-elf` нужно сделать 2 выбора.

## Обязательные выборы

### 1. Заговор волшебника
Нужно выбрать 1 заговор из списка заклинаний волшебника:
- `acid-splash`
- `blade-ward`
- `chill-touch`
- `dancing-lights`
- `fire-bolt`
- `friends`
- `light`
- `mage-hand`
- `mending`
- `message`
- `minor-illusion`
- `poison-spray`
- `prestidigitation`
- `ray-of-frost`
- `shocking-grasp`
- `true-strike`

Формат ответа:
`wizardCantrip: <slug>`

### 2. Дополнительный язык
Помимо `Common` и `Elvish`, нужно выбрать ещё 1 язык:
- `common`
- `dwarvish`
- `elvish`
- `giant`
- `gnomish`
- `goblin`
- `halfling`
- `orc`
- `abyssal`
- `celestial`
- `draconic`
- `deep-speech`
- `infernal`
- `primordial`
- `sylvan`
- `undercommon`

Формат ответа:
`extraLanguage: <slug>`
