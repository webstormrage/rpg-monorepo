# Выборы для high-elf

На этапе выбора расы `high-elf` нужно сделать 2 выбора.

## Обязательные выборы

### 1. Заговор волшебника
Нужно выбрать 1 заговор из списка заклинаний волшебника.

Формат ответа:
`wizardCantrip: <spell>`

### 2. Дополнительный язык
Помимо `Common` и `Elvish`, нужно выбрать ещё 1 язык.

Формат ответа:
`extraLanguage: <language>`
