# Выборы для dragonborn

На этапе выбора расы `dragonborn` нужно сделать 1 выбор.

## Обязательный выбор

### Драконье наследие
Выбери тип дракона. От этого зависят:
- тип урона сопротивления
- форма оружия дыхания
- спасбросок для оружия дыхания

Доступные варианты:
- `white`
- `brass`
- `black`
- `bronze`
- `copper`
- `gold`
- `green`
- `red`
- `silver`
- `blue`

Формат ответа:
`draconicAncestry: <slug>`
