# Выборы для hermit

На этапе выбора `hermit` нужно сделать следующие выборы.

## 1. Язык
Выбери 1 дополнительный язык.

Формат ответа:
`language: <language>`

## 2. Причина уединения
Выбери или опиши причину, по которой персонаж ушёл в отшельничество.

Формат ответа:
`seclusionReason: <slug-or-text>`
