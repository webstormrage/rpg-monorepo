# Выборы для soldier

На этапе выбора `soldier` нужно сделать следующие выборы.

## 1. Игровой набор
Выбери 1 игровой набор.

Формат ответа:
`gamingSet: <set>`

## 2. Военная специализация
Выбери роль, которую персонаж выполнял в армии, из таблицы background.

Формат ответа:
`militarySpecialty: <slug-or-text>`
