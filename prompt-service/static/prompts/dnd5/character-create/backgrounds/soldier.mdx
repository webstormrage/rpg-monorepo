# Выборы для soldier

На этапе выбора `soldier` нужно сделать следующие выборы.

## 1. Игровой набор
Выбери 1 игровой набор:
- `playing-card-set`
- `dice-set`
- `dragonchess-set`
- `three-dragon-ante-set`

Формат ответа:
`gamingSet: <slug>`

## 2. Военная специализация
Выбери роль, которую персонаж выполнял в армии:
- `officer`
- `scout`
- `infantry`
- `cavalry`
- `healer`
- `quartermaster`
- `standard-bearer`
- `support-staff`

Формат ответа:
`militarySpecialty: <slug>`
