# Выборы для entertainer

На этапе выбора `entertainer` нужно сделать следующие выборы.

## 1. Музыкальный инструмент
Выбери 1 музыкальный инструмент.

Формат ответа:
`instrument: <instrument>`

## 2. Стартовый музыкальный инструмент
В снаряжении укажи тот же или другой выбранный инструмент, если это допускает кампания.

Формат ответа:
`startingInstrument: <instrument>`

## 3. Амплуа
Выбери от 1 до 3 сценических амплуа из таблицы background.

Формат ответа:
`routines: [<routine>, ...]`
