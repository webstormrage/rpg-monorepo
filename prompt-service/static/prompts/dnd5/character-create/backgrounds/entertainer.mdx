# Выборы для entertainer

На этапе выбора `entertainer` нужно сделать следующие выборы.

## 1. Музыкальный инструмент
Выбери 1 музыкальный инструмент:
- `bagpipes`
- `drum`
- `dulcimer`
- `flute`
- `horn`
- `lute`
- `lyre`
- `pan-flute`
- `shawm`
- `viol`

Формат ответа:
`instrument: <slug>`

## 2. Стартовый музыкальный инструмент
В снаряжении укажи тот же или другой выбранный инструмент, если это допускает кампания.

Доступные варианты:
- `bagpipes`
- `drum`
- `dulcimer`
- `flute`
- `horn`
- `lute`
- `lyre`
- `pan-flute`
- `shawm`
- `viol`

Формат ответа:
`startingInstrument: <slug>`

## 3. Амплуа
Выбери от 1 до 3 сценических амплуа:
- `acrobat`
- `actor`
- `juggler`
- `musician`
- `singer`
- `fire-eater`
- `poet`
- `storyteller`
- `dancer`
- `jester`

Формат ответа:
`routines: [<slug>, ...]`
