# Выборы для guild-artisan

На этапе выбора `guild-artisan` нужно сделать следующие выборы.

## 1. Ремесленные инструменты
Выбери 1 вид ремесленных инструментов.

Формат ответа:
`toolProficiency: <tool>`

## 2. Язык
Выбери 1 дополнительный язык.

Формат ответа:
`language: <language>`

## 3. Специализация гильдии
Выбери специализацию гильдии из таблицы background.

Формат ответа:
`guildSpecialty: <slug-or-text>`
