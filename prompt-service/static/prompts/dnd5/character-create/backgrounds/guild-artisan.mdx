# Выборы для guild-artisan

На этапе выбора `guild-artisan` нужно сделать следующие выборы.

## 1. Ремесленные инструменты
Выбери 1 вид ремесленных инструментов:
- `alchemists-supplies`
- `brewers-supplies`
- `calligraphers-supplies`
- `carpenters-tools`
- `cartographers-tools`
- `cobblers-tools`
- `cooks-utensils`
- `glassblowers-tools`
- `jewelers-tools`
- `leatherworkers-tools`
- `masons-tools`
- `painters-supplies`
- `potters-tools`
- `smiths-tools`
- `tinkers-tools`
- `weavers-tools`
- `woodcarvers-tools`

Формат ответа:
`toolProficiency: <slug>`

## 2. Язык
Выбери 1 дополнительный язык:
- `common`
- `dwarvish`
- `elvish`
- `giant`
- `gnomish`
- `goblin`
- `halfling`
- `orc`
- `abyssal`
- `celestial`
- `draconic`
- `deep-speech`
- `infernal`
- `primordial`
- `sylvan`
- `undercommon`

Формат ответа:
`language: <slug>`

## 3. Специализация гильдии
Выбери специализацию гильдии:
- `alchemists-and-apothecaries`
- `potters-and-ceramicists`
- `tinsmiths-wireworkers-and-casters`
- `calligraphers-scribes-and-notaries`
- `masons-and-stonecutters`
- `cartographers-surveyors-and-drafters`
- `leatherworkers-furriers-and-tanners`
- `blacksmiths-and-armorers`
- `metallurgists-and-metalworkers`
- `brewers-and-vintners`
- `carpenters-roofers-and-plasterers`
- `cooks-and-bakers`
- `cobblers-and-shoemakers`
- `glassblowers-and-glaziers`
- `joiners-coopers-and-bowyers`
- `shipwrights-and-sailmakers`
- `wagonmakers-and-wheelwrights`
- `weavers-and-dyers`
- `painters-illustrators-and-decorators`
- `jewelers-and-gemcutters`

`guildSpecialty: <slug>`
