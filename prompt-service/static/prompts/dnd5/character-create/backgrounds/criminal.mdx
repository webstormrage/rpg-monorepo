# Выборы для criminal

На этапе выбора `criminal` нужно сделать следующие выборы.

## 1. Игровой набор
Выбери 1 игровой набор:
- `playing-card-set`
- `dice-set`
- `dragonchess-set`
- `three-dragon-ante-set`

Формат ответа:
`gamingSet: <slug>`

## 2. Преступная специализация
Выбери специализацию из таблицы background.

Примеры направлений:
- `blackmailer`
- `burglar`
- `enforcer`
- `fence`
- `highway-robber`
- `hired-killer`
- `pickpocket`
- `smuggler`

Формат ответа:
`criminalSpecialty: <slug>`
