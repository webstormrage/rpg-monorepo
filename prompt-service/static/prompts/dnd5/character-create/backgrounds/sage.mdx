# Выборы для sage

На этапе выбора `sage` нужно сделать следующие выборы.

## 1. Языки
Выбери 2 языка:
- `common`
- `dwarvish`
- `elvish`
- `giant`
- `gnomish`
- `goblin`
- `halfling`
- `orc`
- `abyssal`
- `celestial`
- `draconic`
- `deep-speech`
- `infernal`
- `primordial`
- `sylvan`
- `undercommon`

Формат ответа:
`languages: [<slug>, <slug>]`

## 2. Область исследований
Выбери область знаний из таблицы background:
- `discredited-academic`
- `alchemist`
- `astronomer`
- `librarian`
- `researcher`
- `scribe`
- `wizards-apprentice`
- `professor`

Формат ответа:
`specialty: <slug>`
