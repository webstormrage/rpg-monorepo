# Выборы для acolyte

На этапе выбора `acolyte` нужно сделать следующие выборы.

## 1. Религиозная принадлежность
Выбери:
- конкретное божество: Akadi, Auril, Azuth, Bane, Beshaba, Chauntea, Cyric, Deneir, Eldath, Gond, Helm, Ilmater, Kelemvor, Lathander, Leira, Lliira, Malar, Mask, Mielikki, Milil, Myrkul, Mystra, Oghma, Savras, Selune, Shar, Silvanus, Sune, Talona, Talos, Tempus, Torm, Tymora, Tyr, Umberlee, Waukeen

Формат ответа:
`faith: <name>`

## 2. Языки
Выбери 2 языка: Common, Dwarvish, Elvish, Giant, Gnomish, Goblin, Halfling, Orc, Abyssal, Celestial, Draconic, Deep Speech, Infernal, Primordial, Sylvan, Undercommon.

Формат ответа:
`languages: [<language>, <language>]`
