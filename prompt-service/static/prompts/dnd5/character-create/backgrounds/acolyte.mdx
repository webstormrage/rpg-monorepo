# Выборы для acolyte

На этапе выбора `acolyte` нужно сделать следующие выборы.

## 1. Религиозная принадлежность
Выбери:
- конкретное божество
- пантеон
- или иную религиозную/культовую сущность, если это согласовано с Мастером

Формат ответа:
`faith: <name>`

## 2. Языки
Выбери 2 языка.

Формат ответа:
`languages: [<language>, <language>]`
