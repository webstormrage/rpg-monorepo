# Выборы для outlander

На этапе выбора `outlander` нужно сделать следующие выборы.

## 1. Музыкальный инструмент
Выбери 1 музыкальный инструмент:
- `bagpipes`
- `drum`
- `dulcimer`
- `flute`
- `horn`
- `lute`
- `lyre`
- `pan-flute`
- `shawm`
- `viol`

Формат ответа:
`instrument: <slug>`

## 2. Язык
Выбери 1 дополнительный язык:
- `common`
- `dwarvish`
- `elvish`
- `giant`
- `gnomish`
- `goblin`
- `halfling`
- `orc`
- `abyssal`
- `celestial`
- `draconic`
- `deep-speech`
- `infernal`
- `primordial`
- `sylvan`
- `undercommon`

Формат ответа:
`language: <slug>`

## 3. Происхождение странника
Выбери, откуда именно ты пришёл, из таблицы происхождения/рода занятий background:
- `forester`
- `trapper`
- `homesteader`
- `guide`
- `exile-or-outcast`
- `bounty-hunter`
- `pilgrim`
- `tribal-nomad`
- `hunter-gatherer`
- `tribal-marauder`

Формат ответа:
`origin: <slug>`
