# Выборы для noble

На этапе выбора `noble` нужно сделать следующие выборы.

## 1. Игровой набор
Выбери 1 игровой набор:
- `playing-card-set`
- `dice-set`
- `dragonchess-set`
- `three-dragon-ante-set`

Формат ответа:
`gamingSet: <slug>`

## 2. Язык
Выбери 1 дополнительный язык:
- `common`
- `dwarvish`
- `elvish`
- `giant`
- `gnomish`
- `goblin`
- `halfling`
- `orc`
- `abyssal`
- `celestial`
- `draconic`
- `deep-speech`
- `infernal`
- `primordial`
- `sylvan`
- `undercommon`

Формат ответа:
`language: <slug>`

## 3. Титул
Определи благородный титул персонажа.

Формат ответа:
`title: <text>`
