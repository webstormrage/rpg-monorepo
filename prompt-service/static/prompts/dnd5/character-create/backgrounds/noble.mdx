# Выборы для noble

На этапе выбора `noble` нужно сделать следующие выборы.

## 1. Игровой набор
Выбери 1 игровой набор.

Формат ответа:
`gamingSet: <set>`

## 2. Язык
Выбери 1 дополнительный язык.

Формат ответа:
`language: <language>`

## 3. Титул
Определи благородный титул персонажа.

Формат ответа:
`title: <text>`
