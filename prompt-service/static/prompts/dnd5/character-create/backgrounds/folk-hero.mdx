# Выборы для folk-hero

На этапе выбора `folk-hero` нужно сделать следующие выборы.

## 1. Ремесленные инструменты
Выбери 1 вид ремесленных инструментов:
- `alchemists-supplies`
- `brewers-supplies`
- `calligraphers-supplies`
- `carpenters-tools`
- `cartographers-tools`
- `cobblers-tools`
- `cooks-utensils`
- `glassblowers-tools`
- `jewelers-tools`
- `leatherworkers-tools`
- `masons-tools`
- `painters-supplies`
- `potters-tools`
- `smiths-tools`
- `tinkers-tools`
- `weavers-tools`
- `woodcarvers-tools`

Формат ответа:
`toolProficiency: <slug>`

## 2. Определяющее событие
Выбери событие, которое сделало тебя народным героем.

Формат ответа:
`definingEvent: <slug-or-text>`
