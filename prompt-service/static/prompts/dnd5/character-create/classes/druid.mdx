# Выборы для druid

На 1 уровне для `druid` нужно сделать следующие выборы.

## 1. Навыки
Выбери 2 навыка из списка:
- `perception`
- `survival`
- `arcana`
- `medicine`
- `animal-handling`
- `nature`
- `insight`
- `religion`

Формат ответа:
`skillProficiencies: [<skill>, <skill>]`

## 2. Стартовое снаряжение
Сделай выборы в каждой строке:
- `wooden-shield` или `simple-weapon`
- `scimitar` или `simple-melee-weapon`
- `dungeoneers-pack` или `explorers-pack`

Дополнительно без выбора:
- `druidic-focus`

Формат ответа:
`equipmentChoices: {...}`

## 3. Заговоры
Выбери 2 заговора друида.

Формат ответа:
`cantrips: [<spell>, <spell>]`

## 4. Подготовленные заклинания
Подготовь количество заклинаний, равное:
`wisdomModifier + 1`

Формат ответа:
`preparedSpells: [<spell>, ...]`
