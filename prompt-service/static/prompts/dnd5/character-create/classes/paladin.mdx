# Выборы для paladin

На 1 уровне для `paladin` нужно сделать следующие выборы.

## 1. Навыки
Выбери 2 навыка из списка:
- `athletics`
- `intimidation`
- `medicine`
- `insight`
- `religion`
- `persuasion`

Формат ответа:
`skillProficiencies: [<skill>, <skill>]`

## 2. Стартовое снаряжение
Сделай выборы в каждой строке:
- `martial-weapon-and-shield` или `two-martial-weapons`
- `javelins-5` или `simple-melee-weapon`
- `priests-pack` или `explorers-pack`

Дополнительно без выбора:
- `chain-mail`
- `holy-symbol`

Формат ответа:
`equipmentChoices: {...}`
