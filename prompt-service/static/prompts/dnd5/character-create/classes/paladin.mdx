# Выборы для paladin

На 1 уровне для `paladin` нужно сделать следующие выборы.

## 1. Навыки
Выбери 2 навыка из списка:
- `athletics`
- `intimidation`
- `medicine`
- `insight`
- `religion`
- `persuasion`

Формат ответа:
`skillProficiencies: [<skill>, <skill>]`

## 2. Основной набор оружия
Выбери 2 воинских оружия:
- `battleaxe`
- `flail`
- `glaive`
- `greataxe`
- `greatsword`
- `halberd`
- `lance`
- `longsword`
- `maul`
- `morningstar`
- `pike`
- `rapier`
- `scimitar`
- `shortsword`
- `trident`
- `war-pick`
- `warhammer`
- `whip`
- `blowgun`
- `hand-crossbow`
- `heavy-crossbow`
- `longbow`
- `net`
- `shield`

Формат ответа:
`primaryWeapons: [<slug>, <slug>]`

## 3. Дополнительное оружие
Выбери 1 вариант:
- `javelin`, `javelin`, `javelin`, `javelin`, `javelin`
- `club`
- `dagger`

- `handaxe`
- `javelin`
- `light-hammer`
- `mace`
- `quarterstaff`
- `sickle`
- `spear`

Формат ответа:

## 4. Набор снаряжения
Выбери 1 вариант:
- `priests-pack`
- `explorers-pack`

Формат ответа:
`equipmentPack: <slug>`
