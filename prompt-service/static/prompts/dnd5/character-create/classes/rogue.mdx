# Выборы для rogue

На 1 уровне для `rogue` нужно сделать следующие выборы.

## 1. Навыки
Выбери 4 навыка из списка:
- `acrobatics`
- `athletics`
- `perception`
- `performance`
- `intimidation`
- `sleight-of-hand`
- `deception`
- `insight`
- `investigation`
- `stealth`
- `persuasion`

Формат ответа:
`skillProficiencies: [<skill>, <skill>, <skill>, <skill>]`

## 2. Стартовое снаряжение
Сделай выборы в каждой строке:
- `rapier` или `shortsword`
- `shortbow-and-20-arrows` или `shortsword`
- `burglars-pack` или `dungeoneers-pack` или `explorers-pack`

Дополнительно без выбора:
- `leather-armor`
- `two-daggers`
- `thieves-tools`

Формат ответа:
`equipmentChoices: {...}`

## 3. Экспертиза
Выбери 2 владения для `Expertise`:
- либо 2 навыка
- либо 1 навык и `thieves-tools`

Формат ответа:
`expertise: [<proficiency>, <proficiency>]`
