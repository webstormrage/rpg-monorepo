# Выборы для rogue

На 1 уровне для `rogue` нужно сделать следующие выборы.

## 1. Навыки
Выбери 4 навыка из списка:
- `acrobatics`
- `athletics`
- `perception`
- `performance`
- `intimidation`
- `sleight-of-hand`
- `deception`
- `insight`
- `investigation`
- `stealth`
- `persuasion`

Формат ответа:
`skillProficiencies: [<skill>, <skill>, <skill>, <skill>]`

## 2. Стартовое оружие
Выбери 1 вариант:
- `rapier`
- `shortsword`

Формат ответа:
`startingWeapon: <slug>`

## 3. Дальнобойное или дополнительное оружие
Выбери 1 вариант:
- `shortbow`
- `shortsword`

Формат ответа:
`secondaryWeapon: <slug>`

## 4. Набор снаряжения
Выбери 1 вариант:
- `burglars-pack`
- `dungeoneers-pack`
- `explorers-pack`

Формат ответа:
`equipmentPack: <slug>`

## 5. Экспертиза
Выбери 2 владения для `Expertise`: это должны быть навыки, выбранные в пункте 1, либо `thieves-tools`.
- `acrobatics`
- `athletics`
- `perception`
- `performance`
- `intimidation`
- `sleight-of-hand`
- `deception`
- `insight`
- `investigation`
- `stealth`
- `persuasion`
- `thieves-tools`

Можно выбрать либо 2 навыка из пункта 1, либо 1 навык из пункта 1 и `thieves-tools`.

Формат ответа:
`expertise: [<slug>, <slug>]`
