# Выборы для warlock

На 1 уровне для `warlock` нужно сделать следующие выборы.

## 1. Навыки
Выбери 2 навыка из списка:
- `intimidation`
- `history`
- `arcana`
- `deception`
- `nature`
- `investigation`
- `religion`

Формат ответа:
`skillProficiencies: [<skill>, <skill>]`

## 2. Стартовое снаряжение
Сделай выборы в каждой строке:
- `light-crossbow-and-20-bolts` или `simple-weapon`
- `component-pouch` или `arcane-focus`
- `scholars-pack` или `dungeoneers-pack`

Дополнительно без выбора:
- `leather-armor`
- `simple-weapon`
- `two-daggers`

Формат ответа:
`equipmentChoices: {...}`

## 3. Потусторонний покровитель
Выбери 1 покровителя.

Базовые варианты:
- `archfey`
- `fiend`
- `great-old-one`

Формат ответа:
`otherworldlyPatron: <slug>`

## 4. Заговоры
Выбери 2 заговора колдуна.

Формат ответа:
`cantrips: [<spell>, <spell>]`

## 5. Известные заклинания
Выбери 2 заклинания 1 уровня из списка колдуна.

Формат ответа:
`knownSpells: [<spell>, <spell>]`
