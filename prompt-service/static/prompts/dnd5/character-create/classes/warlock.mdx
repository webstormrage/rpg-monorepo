# Выборы для warlock

На 1 уровне для `warlock` нужно сделать следующие выборы.

## 1. Навыки
Выбери 2 навыка из списка:
- `intimidation`
- `history`
- `arcana`
- `deception`
- `nature`
- `investigation`
- `religion`

Формат ответа:
`skillProficiencies: [<skill>, <skill>]`

## 2. Стартовое оружие
Выбери 1 вариант:
- `light-crossbow`
- `club`
- `dagger`
- `greatclub`
- `handaxe`
- `javelin`
- `light-hammer`
- `mace`
- `quarterstaff`
- `sickle`
- `spear`
- `dart`
- `shortbow`
- `sling`

Формат ответа:
`startingWeapon: <slug>`

## 3. Магический фокус или компоненты
Выбери 1 вариант:
- `component-pouch`
- `arcane-focus`

Формат ответа:
`spellcastingFocus: <slug>`

## 4. Набор снаряжения
Выбери 1 вариант:
- `scholars-pack`
- `dungeoneers-pack`

Формат ответа:
`equipmentPack: <slug>`

## 5. Потусторонний покровитель
Выбери 1 покровителя.

Базовые варианты:
- `archfey`
- `fiend`
- `great-old-one`

Формат ответа:
`otherworldlyPatron: <slug>`

## 6. Заговоры
Выбери 2 заговора колдуна:
- `blade-ward`
- `chill-touch`
- `eldritch-blast`
- `friends`
- `mage-hand`
- `minor-illusion`
- `poison-spray`
- `prestidigitation`
- `true-strike`
- `sword-burst`
- `booming-blade`
- `green-flame-blade`
- `lightning-lure`
- `mind-sliver`

Формат ответа:
`cantrips: [<slug>, <slug>]`

## 7. Известные заклинания
Выбери 2 заклинания 1 уровня из списка колдуна:
- `arms-of-hadar`
- `armor-of-agathys`
- `charm-person`
- `comprehend-languages`
- `expeditious-retreat`
- `hellish-rebuke`
- `hex`
- `illusory-script`
- `protection-from-evil-and-good`
- `unseen-servant`
- `witch-bolt`
- `faerie-fire` (доступно для покровителя `archfey`)
- `sleep` (доступно для покровителя `archfey`)
- `burning-hands` (доступно для покровителя `fiend`)
- `command` (доступно для покровителя `fiend`)
- `dissonant-whispers` (доступно для покровителя `great-old-one`)
- `tashas-hideous-laughter` (доступно для покровителя `great-old-one`)

Формат ответа:
`knownSpells: [<slug>, <slug>]`
