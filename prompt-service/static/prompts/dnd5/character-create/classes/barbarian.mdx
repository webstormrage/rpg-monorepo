# Выборы для barbarian

На 1 уровне для `barbarian` нужно сделать следующие выборы.

## 1. Навыки
Выбери 2 навыка из списка:
- `athletics`
- `perception`
- `survival`
- `intimidation`
- `nature`
- `animal-handling`

Формат ответа:
`skillProficiencies: [<skill>, <skill>]`

## 2. Основное оружие
Выбери 1 воинское рукопашное оружие:
- `battleaxe`
- `flail`
- `glaive`
- `greataxe`
- `greatsword`
- `halberd`
- `lance`
- `longsword`
- `maul`
- `morningstar`
- `pike`
- `rapier`
- `scimitar`
- `shortsword`
- `trident`
- `war-pick`
- `warhammer`
- `whip`

Формат ответа:
`primaryWeapon: <slug>`

## 3. Дополнительное оружие
Выбери 1 вариант:
- `club`
- `dagger`
- `greatclub`
- `handaxe`, `handaxe`
- `javelin`
- `light-hammer`
- `mace`
- `quarterstaff`
- `sickle`
- `spear`

Формат ответа:
`secondaryWeapon: <slug>`

## 4. Набор снаряжения
Выбери 1 вариант:
- `explorers-pack`
- `starting-gold`

Формат ответа:
`equipmentPack: <slug>`
