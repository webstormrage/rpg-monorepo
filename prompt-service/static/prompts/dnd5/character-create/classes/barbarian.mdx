# Выборы для barbarian

На 1 уровне для `barbarian` нужно сделать следующие выборы.

## 1. Навыки
Выбери 2 навыка из списка:
- `athletics`
- `perception`
- `survival`
- `intimidation`
- `nature`
- `animal-handling`

Формат ответа:
`skillProficiencies: [<skill>, <skill>]`

## 2. Стартовое снаряжение
Сделай выборы в каждой строке:
- `greataxe` или `martial-melee-weapon`
- `handaxes-2` или `simple-weapon`
- `explorers-pack` или купить снаряжение за золото

Формат ответа:
`equipmentChoices: {...}`
