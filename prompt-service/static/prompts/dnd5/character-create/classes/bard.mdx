# Выборы для bard

На 1 уровне для `bard` нужно сделать следующие выборы.

## 1. Инструменты
Выбери 3 музыкальных инструмента.

Формат ответа:
`instrumentProficiencies: [<instrument>, <instrument>, <instrument>]`

## 2. Навыки
Выбери 3 любых навыка.

Формат ответа:
`skillProficiencies: [<skill>, <skill>, <skill>]`

## 3. Стартовое снаряжение
Сделай выборы в каждой строке:
- `rapier` или `longsword` или `simple-weapon`
- `diplomats-pack` или `entertainers-pack`
- `lute` или `other-musical-instrument`

Формат ответа:
`equipmentChoices: {...}`

## 4. Заговоры
Выбери 2 заговора барда.

Формат ответа:
`cantrips: [<spell>, <spell>]`

## 5. Известные заклинания
Выбери 4 заклинания 1 уровня из списка барда.

Формат ответа:
`knownSpells: [<spell>, <spell>, <spell>, <spell>]`
