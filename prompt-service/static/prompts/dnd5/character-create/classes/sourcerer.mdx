# Выборы для sourcerer

На 1 уровне для `sourcerer` нужно сделать следующие выборы.

## 1. Навыки
Выбери 2 навыка из списка:
- `intimidation`
- `arcana`
- `deception`
- `insight`
- `religion`
- `persuasion`

Формат ответа:
`skillProficiencies: [<skill>, <skill>]`

## 2. Стартовое снаряжение
Сделай выборы в каждой строке:
- `light-crossbow-and-20-bolts` или `simple-weapon`
- `component-pouch` или `arcane-focus`
- `dungeoneers-pack` или `explorers-pack`

Дополнительно без выбора:
- `two-daggers`

Формат ответа:
`equipmentChoices: {...}`

## 3. Источник магии
Выбери 1 магическое происхождение.

Базовые варианты:
- `draconic-bloodline`
- `wild-magic`

Формат ответа:
`sorcerousOrigin: <slug>`

## 4. Заговоры
Выбери 4 заговора чародея.

Формат ответа:
`cantrips: [<spell>, <spell>, <spell>, <spell>]`

## 5. Известные заклинания
Выбери 2 заклинания 1 уровня из списка чародея.

Формат ответа:
`knownSpells: [<spell>, <spell>]`
