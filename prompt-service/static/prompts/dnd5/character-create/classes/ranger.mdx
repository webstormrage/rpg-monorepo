# Выборы для ranger

На 1 уровне для `ranger` нужно сделать следующие выборы.

## 1. Навыки
Выбери 3 навыка из списка:
- `athletics`
- `perception`
- `survival`
- `nature`
- `insight`
- `investigation`
- `stealth`
- `animal-handling`

Формат ответа:
`skillProficiencies: [<skill>, <skill>, <skill>]`

## 2. Стартовая броня
Выбери 1 вариант:
- `scale-mail`
- `leather-armor`

Формат ответа:
`startingArmor: <slug>`

## 3. Оружие ближнего боя
Выбери 2 простых оружия ближнего боя:
- `club`
- `dagger`
- `greatclub`
- `handaxe`
- `javelin`
- `light-hammer`
- `mace`
- `quarterstaff`
- `sickle`
- `spear`

Формат ответа:
`meleeWeapons: [<slug>, <slug>]`

## 4. Набор снаряжения
Выбери 1 вариант:
- `dungeoneers-pack`
- `explorers-pack`

Формат ответа:
`equipmentPack: <slug>`

## 5. Избранный враг
Выбери  один тип существ из списка:
- `aberration`
- `beast`
- `celestial`
- `construct`
- `dragon`
- `elemental`
- `fey`
- `fiend`
- `giant`
- `humanoid`
- `monstrosity`
- `ooze`
- `plant`
- `undead`

Формат ответа:
`favoredEnemy: <creature-type-slug-or-humanoid-races-list>`

## 7. Известная местность
Выбери 1 тип местности:
- `arctic`
- `coast`
- `desert`
- `forest`
- `grassland`
- `mountain`
- `swamp`
- `underdark`

Формат ответа:
`naturalExplorer: <terrain>`
