# Выборы для ranger

На 1 уровне для `ranger` нужно сделать следующие выборы.

## 1. Навыки
Выбери 3 навыка из списка:
- `athletics`
- `perception`
- `survival`
- `nature`
- `insight`
- `investigation`
- `stealth`
- `animal-handling`

Формат ответа:
`skillProficiencies: [<skill>, <skill>, <skill>]`

## 2. Стартовое снаряжение
Сделай выборы в каждой строке:
- `scale-mail` или `leather-armor`
- `two-shortswords` или `two-simple-melee-weapons`
- `dungeoneers-pack` или `explorers-pack`

Дополнительно без выбора:
- `longbow-and-20-arrows`

Формат ответа:
`equipmentChoices: {...}`

## 3. Избранный враг
Выбери 1 вариант:
- один тип существ
- или две гуманоидные расы

Формат ответа:
`favoredEnemy: <type-or-list>`

## 4. Известная местность
Выбери 1 тип местности:
- `arctic`
- `coast`
- `desert`
- `forest`
- `grassland`
- `mountain`
- `swamp`
- `underdark`

Формат ответа:
`naturalExplorer: <terrain>`
