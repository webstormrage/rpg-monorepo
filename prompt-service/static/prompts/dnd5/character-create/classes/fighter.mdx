# Выборы для fighter

На 1 уровне для `fighter` нужно сделать следующие выборы.

## 1. Навыки
Выбери 2 навыка из списка:
- `acrobatics`
- `athletics`
- `perception`
- `survival`
- `intimidation`
- `history`
- `insight`
- `animal-handling`

Формат ответа:
`skillProficiencies: [<skill>, <skill>]`

## 2. Стартовое снаряжение
Сделай выборы в каждой строке:
- `chain-mail` или `leather-armor-longbow-20-arrows`
- `martial-weapon-and-shield` или `two-martial-weapons`
- `light-crossbow-and-20-bolts` или `two-handaxes`
- `dungeoneers-pack` или `explorers-pack`

Формат ответа:
`equipmentChoices: {...}`

## 3. Боевой стиль
Выбери 1 боевой стиль.

Основные варианты:
- `archery`
- `defense`
- `dueling`
- `great-weapon-fighting`
- `protection`
- `two-weapon-fighting`

Если в проекте используются опциональные стили из исходного файла, можно выбрать их вместо базовых.

Формат ответа:
`fightingStyle: <slug>`
