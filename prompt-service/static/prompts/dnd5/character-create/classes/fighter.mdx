# Выборы для fighter

На 1 уровне для `fighter` нужно сделать следующие выборы.

## 1. Навыки
Выбери 2 навыка из списка:
- `acrobatics`
- `athletics`
- `perception`
- `survival`
- `intimidation`
- `history`
- `insight`
- `animal-handling`

Формат ответа:
`skillProficiencies: [<skill>, <skill>]`

## 2. Стартовая броня
Выбери 1 вариант:
- `chain-mail`
- `leather-armor`, `longbow`

Формат ответа:
`startingArmor: <slug>`

## 3. Основной набор оружия
Выбери 2 воинских оружия:
- `battleaxe`
- `flail`
- `glaive`
- `greataxe`
- `greatsword`
- `halberd`
- `lance`
- `longsword`
- `maul`
- `morningstar`
- `pike`
- `rapier`
- `scimitar`
- `shortsword`
- `trident`
- `war-pick`
- `warhammer`
- `whip`
- `blowgun`
- `hand-crossbow`
- `heavy-crossbow`
- `longbow`
- `net`
- `shield`

Формат ответа:
`primaryWeapons: [<slug>, <slug>]`

## 4. Дополнительное оружие
Выбери 1 вариант:
- `light-crossbow`
- `handaxe`, `handaxe`

Формат ответа:
`secondaryWeaponSet: <slug>`

## 5. Набор снаряжения
Выбери 1 вариант:
- `dungeoneers-pack`
- `explorers-pack`

Формат ответа:
`equipmentPack: <slug>`

## 6. Боевой стиль
Выбери 1 боевой стиль.

Основные варианты:
- `archery`
- `defense`
- `dueling`
- `great-weapon-fighting`
- `protection`
- `two-weapon-fighting`

Формат ответа:
`fightingStyle: <slug>`
