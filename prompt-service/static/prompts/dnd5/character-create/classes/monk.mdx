# Выборы для monk

На 1 уровне для `monk` нужно сделать следующие выборы.

## 1. Инструмент
Выбери 1 инструмент ремесленника или 1 музыкальный инструмент.

Инструменты ремесленника:
- `alchemists-supplies`
- `brewers-supplies`
- `calligraphers-supplies`
- `carpenters-tools`
- `cartographers-tools`
- `cobblers-tools`
- `cooks-utensils`
- `glassblowers-tools`
- `jewelers-tools`
- `leatherworkers-tools`
- `masons-tools`
- `painters-supplies`
- `potters-tools`
- `smiths-tools`
- `tinkers-tools`
- `weavers-tools`
- `woodcarvers-tools`

Музыкальные инструменты:
- `bagpipes`
- `drum`
- `dulcimer`
- `flute`
- `horn`
- `lute`
- `lyre`
- `pan-flute`
- `shawm`
- `viol`

Формат ответа:
`toolChoice: <slug>`

## 2. Навыки
Выбери 2 навыка из списка:
- `acrobatics`
- `athletics`
- `history`
- `insight`
- `religion`
- `stealth`

Формат ответа:
`skillProficiencies: [<skill>, <skill>]`

## 3. Стартовое оружие
Выбери 1 вариант:
- `shortsword`
- `simple-weapon`

Формат ответа:
`startingWeapon: <slug>`

## 4. Набор снаряжения
Выбери 1 вариант:
- `dungeoneers-pack`
- `explorers-pack`

Формат ответа:
`equipmentPack: <slug>`
