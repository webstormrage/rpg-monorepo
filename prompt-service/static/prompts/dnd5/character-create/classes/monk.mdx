# Выборы для monk

На 1 уровне для `monk` нужно сделать следующие выборы.

## 1. Инструмент
Выбери 1 вариант:
- `artisans-tools`
- `musical-instrument`

Формат ответа:
`toolChoice: <type>`

## 2. Навыки
Выбери 2 навыка из списка:
- `acrobatics`
- `athletics`
- `history`
- `insight`
- `religion`
- `stealth`

Формат ответа:
`skillProficiencies: [<skill>, <skill>]`

## 3. Стартовое снаряжение
Сделай выборы в каждой строке:
- `shortsword` или `simple-weapon`
- `dungeoneers-pack` или `explorers-pack`

Дополнительно без выбора:
- `10-darts`

Формат ответа:
`equipmentChoices: {...}`
