# Выборы для wizzard

На 1 уровне для `wizzard` нужно сделать следующие выборы.

## 1. Навыки
Выбери 2 навыка из списка:
- `history`
- `arcana`
- `medicine`
- `insight`
- `investigation`
- `religion`

Формат ответа:
`skillProficiencies: [<skill>, <skill>]`

## 2. Стартовое снаряжение
Сделай выборы в каждой строке:
- `quarterstaff` или `dagger`
- `component-pouch` или `arcane-focus`
- `scholars-pack` или `explorers-pack`

Дополнительно без выбора:
- `spellbook`

Формат ответа:
`equipmentChoices: {...}`

## 3. Заговоры
Выбери 3 заговора волшебника.

Формат ответа:
`cantrips: [<spell>, <spell>, <spell>]`

## 4. Книга заклинаний
Выбери 6 заклинаний 1 уровня для начальной книги заклинаний.

Формат ответа:
`spellbookSpells: [<spell>, <spell>, <spell>, <spell>, <spell>, <spell>]`

## 5. Подготовленные заклинания
Подготовь количество заклинаний, равное:
`intelligenceModifier + 1`

Формат ответа:
`preparedSpells: [<spell>, ...]`
