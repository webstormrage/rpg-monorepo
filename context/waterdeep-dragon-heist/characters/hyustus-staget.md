# Hyustus Staget
*Средний гуманоид (человек-иллусканец), законно-добрый*

## Образ
Строгий капитан City Watch с бюрократичным и дисциплинированным подходом. Говорит коротко и по делу.

## Атрибуты
- **Класс Доспеха**: по `veteran`
- **Хиты**: по `veteran`
- **Скорость**: 30 фт.
- **СИЛ / ЛОВ / ТЕЛ / ИНТ / МДР / ХАР**: по `veteran`
- **Навыки**: служебная тактика, допрос, контроль порядка
- **Чувства**: пассивная Внимательность по `veteran`
- **Языки**: Common
- **Опасность**: 3 (ориентир `veteran`)
- **Специализация**: командир городского патруля
- **Снаряжение**: форма и вооружение капитана Watch

## Известные факты
Ответственный за порядок в Dock Ward, знает криминальную обстановку района.

## Бэкграунд
Опытный офицер с репутацией прагматика; предпочитает минимизировать городское насилие административными методами.
