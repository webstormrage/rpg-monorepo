# Nihiloor
*Средний аберрация (mind flayer / illithid), мировоззрение по базовому статблоку*

## Образ
Холодный, интеллектуально превосходящий собеседников манипулятор. Предпочитает действовать через контроль и посредников.

## Атрибуты
- **Класс Доспеха**: по `mind flayer`
- **Хиты**: по `mind flayer`
- **Скорость**: по `mind flayer`
- **СИЛ / ЛОВ / ТЕЛ / ИНТ / МДР / ХАР**: по `mind flayer`
- **Навыки**: высокий интеллект, псионический контроль, скрытое управление агентами
- **Чувства**: по `mind flayer`
- **Языки**: по `mind flayer`
- **Опасность**: 7 (2,900 XP) ориентир `mind flayer`
- **Специализация**: псионик-контроллер, работа через intellect devourers
- **Снаряжение**: артефактный ключ-«каменный глаз» (сюжетный)

## Известные факты
Работает в интересах Xanathar, занимается созданием и применением intellect devourers для шпионажа и подмены людей.

## Бэкграунд
Иллитид с собственными долгосрочными планами, использующий криминальные структуры как прикрытие.
