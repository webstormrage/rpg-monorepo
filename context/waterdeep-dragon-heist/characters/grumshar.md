# Grum'shar
*Средний гуманоид (полуорк), хаотично-злой*

## Образ
Жестокий и амбициозный силовик Xanathar Guild, склонный демонстрировать власть через запугивание.

## Атрибуты
- **Класс Доспеха**: по `apprentice wizard`
- **Хиты**: по `apprentice wizard`
- **Скорость**: 30 фт.
- **СИЛ / ЛОВ / ТЕЛ / ИНТ / МДР / ХАР**: по `apprentice wizard`
- **Навыки**: аркана базового мага, силовое давление
- **Чувства**: darkvision 60 фт.
- **Языки**: Common, Orc
- **Опасность**: ориентир `apprentice wizard`
- **Специализация**: контроль пространства и урон через низкоуровневые заклинания
- **Снаряжение**: spellbook, базовое вооружение
- **Заклинания (из книги)**: `burning hands`, `disguise self`, `false life`, `shield`, `unseen servant`, `witch bolt`
- **Особенность расы**: `Relentless Endurance` (1/long rest)

## Известные факты
Руководит локальной ячейкой Xanathar Guild в канализационных районах.

## Бэкграунд
Карьерный криминальный лидер низшего звена, стремящийся укрепить статус внутри организации.
