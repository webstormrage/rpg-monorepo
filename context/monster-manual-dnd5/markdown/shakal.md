другие животные
В связи с ограничениями по объёму, книга не может
перечислить абсолютно всех животных, которые могут
встретиться в вашей кампании. Тем не менее, вы легко
можете использовать параметры одного животного для
другого, похожего на него. Например, вы можете исполь-
зовать параметры пантеры для ягуара, гигантского козла
для буйвола, и сокола для ястреба.
